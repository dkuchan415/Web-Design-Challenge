Latitude Analysis Dashboard with Attitude

Summary: Latitude vs. X

This project examines several weather metrics in cities across the globe with the goal of determining if a trend exists when compared to a city's latitude. The data is from over 500 cities in the OpenWeatherMap API, all observed at the same date and time.

With the API data, four correlation graphs were generated using Matplotlib. These look at temperature, humidity, cloudiness, and wind speed measured against the latitude of the city. Each page contains the graph, as well as a brief explanation of any meaningful observations from the data.
